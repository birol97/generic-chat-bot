## Project: Sui Blockchain Transaction Explainer

### Project Description
This project focuses on retrieving on-chain transaction data from the Sui blockchain and generating meaningful, human-readable explanations of blockchain transactions.

### Key Features
- Non-technical, manually generated transaction summaries
- AI-generated transaction explanations using large language models
- Designed for users with no blockchain background

### Technologies Used
- TypeScript (Frontend)
- GraphQL (Fetching transaction details)
- OpenAI API (AI-generated summaries)

### Domain
Blockchain 路 Web3 路 AI-assisted data interpretation

## Project: Smart Traffic Light System Using Google Maps

### Project Description
An Internet of Things (IoT) project that captures real-time traffic congestion data and dynamically adjusts traffic light timings based on congestion levels.

### Key Features
- Lane-level congestion detection at busy intersections
- Adaptive traffic light control based on congestion rates
- Ambulance emergency module to prioritize emergency vehicles

### Technologies Used
- Python (Congestion detection and control algorithms)
- MQTT (Message broker between hardware and software)
- Raspberry Pi (Hardware interface simulating traffic lights)
- Selenium & OpenCV (Traffic data extraction and lane congestion detection)

### Domain
Internet of Things (IoT) 路 Computer Vision 路 Smart Cities
